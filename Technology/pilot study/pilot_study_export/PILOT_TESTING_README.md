## ğŸ¯ Key Success Factors

### Do These Things Well:

1. **Stay Organized**
   - Follow checklists religiously
   - Label everything with participant IDs
   - Back up data immediately after each session

2. **Maintain Neutrality**
   - Don't judge poetry quality
   - Don't provide creative input
   - Let AI be the partner, you're just the facilitator

3. **Document Everything**
   - Detailed observation notes
   - Exact quotes from participants
   - All technical issues
   - Your own reflections

4. **Be Consistent**
   - Use same scripts for each participant (adjusted for condition)
   - Same session duration
   - Same environment
   - Same interview questions

5. **Back Up Data**
   - 3 locations minimum
   - Immediately after each session
   - Verify backup integrity
